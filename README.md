# BombAdventures
A puzzle-action tile-based game utilising a pseudo-3d viewpoint. Requires python 2.7.X and appropriate pygame version.
## Features
More than 64 Lovely Levels  
5 Bosses (and counting)  
Saving (must beat world to save)
## Controls
WASD and spacebar
